1. پێشەکییەک بۆ پایتۆن

پایتۆن زمانێکی پرۆگرامسازی ئاست-بەرز، لێکدراوە (interpreted)، و فرە-مەبەستە. لەلایەن گویدۆ ڤان ڕۆسەمەوە دروستکراوە و یەکەمجار لە ساڵی 1991 بڵاوکرایەوە. فەلسەفەی دیزاینی پایتۆن جەخت لەسەر خوێندنەوەی ئاسانی کۆد دەکاتەوە، بە بەکارهێنانی بۆشایی سپی (whitespace)ی بەرچاو.

بۆچی پایتۆن؟

  • ئاسان بۆ فێربوون: سینتاکسەکەی (syntax) نزیکە لە زمانی ئینگلیزی، ئەمەش وایکردووە بۆ سەرەتاکاران زۆر گونجاو بێت.
  • فرە-پلاتفۆرم: لەسەر ویندۆز، ماک، لینوکس، و سیستەمەکانی تر کاردەکات.
  • کۆمەڵگەی گەورە: خاوەنی کۆمەڵگەیەکی گەورە و چالاکی گەشەپێدەرانە، کە ئەمەش واتای بوونی کتێبخانە (libraries) و چوارچێوەی (frameworks) زۆرە.
  • فرە-مەبەست: بۆ گەشەپێدانی وێب، زانستی داتا، زیرەکی دەستکرد، ئۆتۆمەیشن، و زۆری تر بەکاردێت.

2. دامەزراندنی پایتۆن

بۆ دەستپێکردن، پێویستە پایتۆن لەسەر کۆمپیوتەرەکەت دابمەزرێنیت.

  1. سەردانی ماڵپەڕی فەرمی پایتۆن بکە: python.org/downloads
  2. نوێترین وەشانی پایتۆن (بۆ نموونە Python 3.11 یان نوێتر) دابگرە.
  3. لەکاتی دامەزراندن لەسەر ویندۆز، دڵنیابە کە نیشانەی "Add Python to PATH" چالاک دەکەیت. ئەمە زۆر گرنگە بۆ ئەوەی بتوانیت پایتۆن لە (Command Prompt)ەوە بەکاربهێنیت.
  4. بۆ دڵنیابوونەوە لەوەی کە بە سەرکەوتوویی دامەزراوە، (Command Prompt) یان (Terminal) بکەرەوە و بنووسە:
python --version

یان ئەگەر لەسەر ماک یان لینوکس بیت، ڕەنگە پێویست بکات بنووسیت: 

python3 --version

3. جێبەجێکردنی پرۆگرامەکانی پایتۆن

دوو ڕێگەی سەرەکی هەیە بۆ جێبەجێکردنی کۆدی پایتۆن:

أ. دۆخی کارلێککار (Interactive Mode - REPL)

تەنها (Terminal) یان (CMD) بکەرەوە و بنووسە python (یان python3) و Enter دابگرە. ئەمە دەتەوێتە ناو شێڵ (shell)ی پایتۆن، کە دەتوانیت کۆد دێڕ بە دێڕ بنووسیت و ئەنجامەکەی ڕاستەوخۆ ببینیت. 

>>> print("سڵاو، جیهان!")
سڵاو، جیهان!
>>> 2 + 3
5

بۆ دەرچوون لەم دۆخە، بنووسە exit().

ب. دۆخی سکریپت (Script Mode)

دەتوانیت کۆدەکانت لەناو فایلێکدا هەڵبگریت کە پاشگری .py هەیە (بۆ نموونە my_program.py). پاشان فایلەکە لە ڕێگەی تێرمیناڵەوە جێبەجێ بکەیت.

فایلی my_program.py:

# This is my first Python program
name = "ئاریا"
print(f"سڵاو، {name}!")

بۆ جێبەجێکردنی لە تێرمیناڵ:

python my_program.py

ئەنجام:

سڵاو، ئاریا!

5.1. گۆڕاو (Variable)

گۆڕاوەکان وەکو دەفر یان کۆنتەینەر وان بۆ هەڵگرتنی بەهاکانی داتا. لە پایتۆندا، پێویست ناکات جۆری گۆڕاوەکە پێشوەختە دیاری بکەیت. تەنها ناوێک و بەهایەکی پێدەدەیت. 

# x گۆڕاوێکە کە ژمارەی تەواو (integer) هەڵدەگرێت
x = 10

# name گۆڕاوێکە کە ستริง (string) هەڵدەگرێت
name = "هێمن"

# pi گۆڕاوێکە کە ژمارەی خاڵدار (float) هەڵدەگرێت
pi = 3.14

# is_student گۆڕاوێکە کە بووڵیەن (boolean) هەڵدەگرێت
is_student = True

print(x)
print(name)

یاساکانی ناونانی گۆڕاو

  • پێویستە بە پیت یان هێمای ژێرەوە (_) دەست پێبکات.
  • ناتوانێت بە ژمارە دەست پێبکات.
  • تەنها دەتوانێت پیت (a-z, A-Z)، ژمارە (0-9)، و هێمای ژێرەوە (_) لەخۆبگرێت.
  • ناوەکان هەستیارن بە پیتی گەورە و بچووک (age جیاوازە لە Age).

4. پایتۆن 2 بەرامبەر پایتۆن 3

دوو وەشانی سەرەکی پایتۆن هەبوون: پایتۆن 2 و پایتۆن 3.

پایتۆن 2: وەشانی کۆنترە. پشتگیری فەرمی بۆ پایتۆن 2 لە 1ی کانوونی دووەمی 2020 کۆتایی هات. نابێت هیچ پڕۆژەیەکی نوێ بە پایتۆن 2 دەست پێبکەیت.

پایتۆن 3: وەشانی ئێستا و داهاتووی پایتۆنە. هەموو کتێبخانە و ئامرازە نوێیەکان پشتگیری پایتۆن 3 دەکەن. ئەم فێرکارییە بە تەواوی لەسەر پایتۆن 3 دەبێت.

هەندێک جیاوازی سەرەکی (بۆ نموونە): 

# Python 2
print "Hello"

# Python 3
print("Hello")

5. بنەماکانی کارکردن لەگەڵ پایتۆن

لەم بەشەدا، چەند چەمکێکی بنەڕەتی دەبینین کە بناغەی پرۆگرامسازی پایتۆنن.

5.2. دەربڕین و ستەیتمێنت (Expressions and Statements)

دەربڕین (Expression): پارچە کۆدێکە کە ئەنجامێکی (بەهایەکی) هەیە. بۆ نموونە 2 + 3 دەربڕینێکە کە ئەنجامەکەی 5.

ستەیتمێنت (Statement): کردارێک یان فرمانێکە کە پایتۆن جێبەجێی دەکات. بۆ نموونە x = 5 ستەیتمێنتێکە کە بەهای 5 دەداتە پاڵ گۆڕاوی x. 

# 10 + 5 دەربڕینە
x = 10 + 5  # x = 15 ستەیتمێنتە

print(x)    # print(x) ستەیتمێنتە

5.3. کۆمێنت (Comment)

کۆمێنتەکان بۆ ڕوونکردنەوەی کۆدەکان بەکاردێن و لەلایەن پایتۆنەوە جێبەجێ ناکرێن. 

# ئەمە کۆمێنتێکی یەک-دێڕییە
print("سڵاو!")  # ئەم کۆمێنتە لە کۆتایی دێڕەکەیە

"""
ئەمە کۆمێنتێکی
فرە-دێڕییە
کە زۆرجار وەک docstring بەکاردێت
"""
print("کۆمێنتەکان گرنگن")

5.4. بۆشایی (Indentation)

لە پایتۆندا، بۆشایی (زۆربەی کات 4 سپەیس) زۆر گرنگە. لەبری بەکارهێنانی کەوانەی {}، پایتۆن بۆشایی بەکاردەهێنێت بۆ دیاریکردنی بلۆکی کۆدەکان (بۆ نموونە ناو if، for، def). 

x = 10

if x > 5:
    print("x گەورەترە لە 5")  # ئەم دێڕە 4 سپەیسی هەیە
    print("ئەمەش هەر لەناو بلۆکی if دایە")
else:
    print("x گچکەترە یان یەکسانە بە 5")

print("ئەمە لە دەرەوەی بلۆکی if ە")

6. جۆرەکانی داتا (Data Types)

پایتۆن چەندین جۆری داتای ئامادەکراوی هەیە. هەندێک لە گرنگترینیان:

  • جۆری دەق (Text Type): str
  • جۆری ژمارەیی (Numeric Types): int, float, complex
  • جۆری ڕیزبەندی (Sequence Types): list, tuple, range
  • جۆری نەخشە (Mapping Type): dict
  • جۆری سێت (Set Types): set, frozenset
  • جۆری بووڵیەن (Boolean Type): bool

دەتوانیت فانکشنی type() بەکاربهێنیت بۆ زانینی جۆری هەر گۆڕاوێک: 

x = 5
y = "سڵاو"
z = 3.14
a = True
b = ["a", "b"]

print(type(x))  # 
print(type(y))  # 
print(type(z))  # 
print(type(a))  # 
print(type(b))  #

7. کردارەکان (Operators)

کردارەکان سیمبوڵی تایبەتن کە بۆ ئەنجامدانی کردار لەسەر گۆڕاوەکان و بەهاکان بەکاردێن.

7.0.1. کرداری دانان (Assignment)

بۆ دانانی بەها بۆ گۆڕاوەکان بەکاردێت. 

x = 5     # دانان
x += 3    # یەکسانە بە: x = x + 3
x -= 2    # یەکسانە بە: x = x - 2
x *= 4    # یەکسانە بە: x = x * 4
x /= 2    # یەکسانە بە: x = x / 2
x %= 3    # یەکسانە بە: x = x % 3 (ماوە)
x //= 2   # یەکسانە بە: x = x // 2 (دابەشکردنی تەواو)
x **= 3   # یەکسانە بە: x = x ** 3 (توان)

7.0.2. کرداری ژمێریاری (Arithmetic)

x = 10
y = 3

print(x + y)  # کۆکردنەوە (13)
print(x - y)  # لێدەرکردن (7)
print(x * y)  # لێکدان (30)
print(x / y)  # دابەشکردن (3.33...)
print(x % y)  # ماوەی دابەشکردن (1)
print(x ** y) # توان (1000)
print(x // y) # دابەشکردنی تەواو - بێ خاڵ (3)

7.0.3. کرداری بەراوردکاری (Comparison)

ئەنجامی True یان False دەگەڕێننەوە. 

x = 5
y = 8

print(x == y) # یەکسانە؟ (False)
print(x != y) # یەکسان نییە؟ (True)
print(x > y)  # گەورەترە؟ (False)
print(x < y)  # گچکەترە؟ (True)
print(x >= y) # گەورەترە یان یەکسانە؟ (False)
print(x <= y) # گچکەترە یان یەکسانە؟ (True)

7.0.4. کرداری لۆجیکی (Boolean)

x = True
y = False

print(x and y) # ئەگەر هەردووکیان True بن، True دەگەڕێنێتەوە (False)
print(x or y)  # ئەگەر یەکێکیان True بێت، True دەگەڕێنێتەوە (True)
print(not x)   # پێچەوانەی بەهاکە دەگەڕێنێتەوە (False)

7.0.5. کرداری Bitwise

ئەم کردارانە لەسەر ئاستی بیت (binary) کاردەکەن. (بۆ ئاستی پێشکەوتوو). 

x = 6  # (0110)
y = 2  # (0010)

print(x & y) # AND (0010 -> 2)
print(x | y) # OR  (0110 -> 6)
print(x ^ y) # XOR (0100 -> 4)
print(~x)    # NOT (-0111 -> -7)
print(x << 1) # Shift left (1100 -> 12)
print(x >> 1) # Shift right (0011 -> 3)

7.0.6. is و in

in (Membership): پشکنین دەکات ئایا بەهایەک لەناو ڕیزبەندییەکدا هەیە.

is (Identity): پشکنین دەکات ئایا دوو گۆڕاو ئاماژەن بۆ *هەمان* ئۆبجێکت لە بیرگە (memory). 

# 'in'
fruits = ["apple", "banana"]
print("apple" in fruits)  # True
print("orange" not in fruits) # True

# 'is'
x = ["a", "b"]
y = ["a", "b"]
z = x

print(x == y) # True (چونكە بەهاکان یەکسانن)
print(x is y) # False (چونكە دوو ئۆبجێکتی جیاوازن لە بیرگە)
print(x is z) # True (چونكە z ئاماژەیە بۆ هەمان ئۆبجێکتی x)

8. کرداری سێیانی (Ternary Operator)

ڕێگەیەکی کورتکراوەیە بۆ نووسینی ڕستەی if...else. 

# شێوازی ئاسایی
age = 20
if age >= 18:
    status = "پێگەیشتوو"
else:
    status = "منداڵ"
print(status)

# شێوازی کرداری سێیانی
status = "پێگەیشتوو" if age >= 18 else "منداڵ"
print(status)

9. ستริงەکان (Strings)

ڕیزێک لە پیتەکانن کە لەناو ' ' یان " " دادەنرێن. 

s = "سڵاو، جیهان!"

# بڕین (Slicing)
print(s[0])      # 'س'
print(s[7:12])   # 'جیهان'

# مێتۆدەکان
print(s.upper())      # "سڵاو، جیهان!" (بە پیتی گەورە، ئەگەر ئینگلیزی بێت)
print(s.lower())      # "سڵاو، جیهان!" (بە پیتی بچووک، ئەگەر ئینگلیزی بێت)
print(len(s))         # 13 (درێژی ستริง)

# f-Strings (بۆ تێکەڵکردنی گۆڕاو)
name = "ئاکام"
age = 25
greeting = f"من ناوم {name}ە و تەمەنم {age} ساڵە."
print(greeting)

10. بووڵیەنەکان (Booleans)

تەنها دوو بەهایان هەیە: True (ڕاست) و False (هەڵە). 

is_active = True
is_admin = False

print(5 > 3)    # True
print(10 == 20) # False

if is_active:
    print("بەکارهێنەر چالاکە")

11. ژمارەکان (Numbers)

پایتۆن سێ جۆر ژمارەی سەرەکی هەیە.

11.0.1. ژمارەی تەواو (Integer)

ژمارەی تەواون، پۆزەتیڤ یان نێگەتیڤ، بێ خاڵی دەیی. 

x = 10
y = -300
z = 123456789
print(type(x)) #

11.0.2. ژمارەی خاڵدار (Floating point)

ژمارەن کە خاڵی دەییان تێدایە. 

x = 3.14
y = -0.5
z = 1.0
print(type(x)) #

11.0.3. ژمارەی ئاڵۆز (Complex)

ژمارەن کە بەشی خەیاڵییان هەیە و بە j نیشانە دەکرێن. 

x = 3 + 5j
y = -2j
print(type(x)) #

11.0.4. کردارە ژمێریارییەکان لەسەر ژمارە

هەموو کردارە ژمێریارییەکان (کە لە بەشی 7.0.2 باسکرا) دەتوانرێت لەسەر ژمارەکان جێبەجێ بکرێن.

11.0.5. فانکشنی ئامادەکراو (Built-in)

print(abs(-7.5))   # 7.5 (ڕەها)
print(round(5.8))  # 6 (نزیککردنەوە)
print(round(5.3))  # 5
print(pow(2, 4))   # 16 (2 بە توانی 4)

12. نەگۆڕەکان (Constants)

پایتۆن بە شێوەیەکی فەرمی "نەگۆڕ"ی نییە (واتە ناتوانرێت بەهای بگۆڕدرێت). بەڵام بەپێی ڕێککەوتن، پرۆگرامەران ناوی گۆڕاوە نەگۆڕەکان بە پیتی گەورە (UPPERCASE) دەنووسن بۆ ئەوەی ئاماژە بێت کە نابێت بگۆڕدرێت. 

PI = 3.14159
MAX_CONNECTIONS = 10

# هەرچەندە لە ڕووی تەکنیکییەوە دەتوانیت بیگۆڕیت، بەڵام نابێت بیکەیت

13. Enums

class Car:
    # Constructor (دروستکەر)
    def __init__(self, brand, model, year):
        self.brand = brand
        self.model = model
        self.year = year

    def stop_engine(self):
        self.is_running = False
        print("ئۆتۆمبێلەکە کوژایەوە")

31. Recursion

کاتێک ڕوودەدات کە فانکشنێک خۆی بانگ بکاتەوە. پێویستی بە مەرجی وەستان (base case) هەیە بۆ ئەوەی نەکەوێتە بازنەی بێ کۆتا. 

# نموونەی فاکتۆریال (5! = 5*4*3*2*1)
def factorial(n):
    # Base case (مەرجی وەستان)
    if n == 1:
        return 1
    # Recursive case (بانگکردنەوەی خۆی)
    else:
        return n * factorial(n - 1)

print(factorial(5)) # 120

33. Closures

Closure فانکشنێکی ناوەکییە (nested) کە دەتوانێت دەستی بگات بە گۆڕاوەکانی فانکشنی دەرەوەی خۆی، تەنانەت دوای ئەوەی فانکشنی دەرەوە جێبەجێکردنی تەواو بووە. 

def outer_func(x):
    def inner_func(y):
        # x لە scopeـی دەرەوە وەردەگرێت
        return x + y
    
    # فانکشنی ناوەوە دەگەڕێنێتەوە
    return inner_func

# add_five ئێستا closureـێکە کە x=5ـی تێدا هەڵگیراوە
add_five = outer_func(5)

# کاتێک بانگی دەکەین، 5 (x) + 10 (y)
print(add_five(10)) # 15

34. Decorators

Decoratorـەکان ڕێگەیەکن بۆ گۆڕینی یان زیادکردنی کرداری نوێ بۆ فانکشنەکان، بەبێ دەستکاریکردنی کۆدی فانکشنە ئەسڵییەکە. 

def my_decorator(func):
    def wrapper():
        print("شتێک پێش بانگکردنی فانکشنەکە")
        func()
        print("شتێک دوای بانگکردنی فانکشنەکە")
    return wrapper

@my_decorator
def say_whee():
    print("Whee!")

say_whee()
# ئەنجام:
# شتێک پێش بانگکردنی فانکشنەکە
# Whee!
# شتێک دوای بانگکردنی فانکشنەکە

38. هەڵەکان (Exceptions)

    print("ئەم بلۆکە هەمیشە جێبەجێ دەبێت")

41. List comprehensions

ڕێگەیەکی کورت و پوختە بۆ دروستکردنی لیست لەسەر بنەمای لیستێکی تر. 

# شێوازی ئاسایی
squares = []
for x in range(10):
    squares.append(x * x)
print(squares)
# [0, 1, 4, 9, 16, 25, 36, 49, 64, 81]


# شێوازی List Comprehension
squares = [x * x for x in range(10)]
print(squares)
# [0, 1, 4, 9, 16, 25, 36, 49, 64, 81]

# لەگەڵ مەرج
even_squares = [x * x for x in range(10) if x % 2 == 0]
print(even_squares) # [0, 4, 16, 36, 64]

واتای "چەندین شێوە". لە پرۆگرامسازیدا، واتە توانای بەکارهێنانی یەک ناوی فانکشن (یان کردار) بۆ چەندین جۆری داتا. 

# فانکشنی len() نموونەی polymorphismـە

43. Operator Overloading

توانای پێناسەکردنی ئەوەی کە کردارەکان (وەک +, -, *) چۆن کار لەسەر ئۆبجێکتەکانی کلاسەکەی تۆ بکەن. 

class Point:
    def __init__(self, x, y):
        self.x = x
        self.y = y

    # پێناسەکردنی '+' بۆ ئەم کلاسە
    def __add__(self, other):
        return Point(self.x + other.x, self.y + other.y)

    def __str__(self):
        return f"({self.x}, {self.y})"

p1 = Point(1, 2)
p2 = Point(3, 4)

p3 = p1 + p2  # لێرەدا __add__ بانگ دەکرێت
print(p3) # (4, 6)
